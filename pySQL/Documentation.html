About MySQL connector python

This program is designed to help you connect to your MySQL database and perform certain 'Special' actions too.
This version has two interface-

  1. Line interface (Also reffered to as Dev interface)
  2. GUI interface (Also reffered to as Client interface)

GUI interface(Client interface)

Installation steps

Before using this program, there are some requirements needed to be installed, they are

  • Python 3.x installed
  • MySQL server instaled

Python IDLE image

GREAT,now that you have installed MPEL(MySQL Python Easy Launcher), let's take a look at what you should make sure while entering MySQL statements/queries

MySQL statements/queries

Firstly, MySQL statements are used as same as you would in your MySQL client, only

  • Semi-colon(;) is not required when you are entering only 1 statement
  • MySQL help utility will not work (\h in MySQL)

Most output is shown in the output box of GUI interface, while all output is shown in python console of Line interface.

Connector commands

This program also comes with some additional functions (the special actions we talked about in the About section).We call these functions 'Connector commands'.Here are some things you need to look out for when using connector commands

  • All connector commands start with exclamatory mark (!)
  • All connector commands are full-caps/upper-case.
  • Fields are seperated by single space ( ), and are case-sensitive

The connector commands are- (click on the name to get straight to it's details)

  1. help (!HELP)
  2. export (!EXPORT)
  3. sheet names (!SHEETNAMES)
  4. rename sheet (!RENAMESHEET)
  5. export database (!EXPORTDB)
  6. delete sheet (!DELETESHEET)
  7. export new (!EXPORTNEW)
  8. configure (!CONFIG)
  9. version (!VERSION)
  10. export path (!EXPORTPATH)
  11. table view (!TABLEVIEW) (GUI interface)

Let's go through them in detail

  1. Help (!HELP)

    The help command tells about all the connector commands. The help command takes no arguement (data)

    The syntax is - !HELP

  2. Export (!EXPORT)

    The export command exports a table to the excel file 'OutPut.excel'.The database of the table being exported has to be selected before exporting.

    The syntax is - !EXPORT table_name sheet_name - where the 'table_name' has to be replaced by the name of the table you want to export (as in MySQL database) and 'sheet_name' has to be replaced by the name of sheet you want in 'OutPut.excel' file.

    The export command creates a new sheet for every export, please select a different name or use !SHEETNAMES to get names of sheets in the 'OutPut.excel' file. This is strictly advised because your data in the sheet will be overwritten if the same name is used

  3. Sheet names (!SHEETNAMES)

    This command tells you the names of sheets in the 'OutPut.excel' file.

    The syntax - !SHEETNAMES

    It is extremely useful when you have exported many tables, or have forgot wether you have exported a table or not

  4. Rename sheet (!RENAMESHEET)

    Renames a sheet present in 'OutPut.excel'.

    The syntax is - !RENAMESHEET old_sheet_name new_sheet_name - replacing 'old_sheet_name' with the name of the sheet you want to change the name of and 'new_sheet_name' with the name you want for that sheet

  5. Export database

    Exports a complete database to excel file 'OutPut.excel'

    The syntax is - !EXPORTDB db_name - where 'db_name' has to be replaced by the name of database you want to export (sheets are named same as the table names).It is advised to be using the database you want to export

  6. Delete sheet

    It deletes a certain sheet present in 'OutPut.excel'

    The syntax is - !DELETESHEET sheet_name - where 'sheet_name' has to be replaced by the name of sheet you want to delete.

  7. Export to new workbook

    This exports a table to a new workbook. The database of which you are exporting the table has to be in use/ has to be selected

    The syntax is - !EXPORTNEW table_name sheet_name workbook_name - where 'table_name' has to be replaced by the name of table you want to export, 'sheet_name' has to be replaced by the sheet name you want and 'workbook_name' by the name of workbook you want

  8. Configure

    Configures MPEL

    The syntax is - !CONFIG

  9. Version

    This tells the version of MPEL

    The syntax is - !VERSION

  10. Export path

    This returns the path where your export file is present (OutPut.excel)

    The syntax is - !EXPORTPATH

  11. Table view

    This displays data in tabulated form

    The syntax is- !TABLEVIEW table_name - where 'table_name' has to be replaced by the the table name of which you want to see the tabulated form
